Дата последнего обновления: 20.05.18 12:31 PM
Данный сервис находится в активной разработке.
Сервис будет предоставлять возможности интелектуального поиска работы. На основе заданных технологий он будет искать наиболее подходящие вакансии на сайтах работы и предлагать их выборку в порядке спадания % соответствия.
Так же можно будет указать "не показывать вакансии где требует *что-то*" или искать только такие вакансии, где это требуется.
Всего в 1 клик можно будет искать именно подходящие вакансии со всех сайтов поиска работы, без нужды тратить лишнее время.

Сейчас сервис находит свежие вакансии по запросу, указанному в url, на сайте rabota.ua и отображает их требования.
Более менее стабильная работа наблюдается по запросу junior java developer
