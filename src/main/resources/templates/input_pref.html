Данный сервис находится в активной разработке.
Сервис будет предоставлять возможности интелектуального поиска работы. На основе заданных предпочтений он будет искать наиболее подходящие вакансии на сайтах работы и предлагать их выборку в порядке спадания % соответствия.
Так же можно будет указать "не показывать вакансии где требует *что-то*" или искать только такие вакансии, где это требуется.
Всего в 1 клик можно будет найти именно подходящие вакансии с самых крупных сайтов для поиска работы, без нужды тратить лишнее время просматривая неподходящие предложения.


Сейчас более-менее стабильная работа сервиса наблюдается по запросу "Junior Java Developer". Для других запросов результат может быть непредсказуемым, т.к. сервис находится в активной разработке.
