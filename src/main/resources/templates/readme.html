Login page. After successful authentication u'll get a token with some authority(role).
With login and password: admin u'll get ADMIN authority.
With login and password: user u'll get USER authority.

For access control page, u should have ADMIN authority.
For access simple hello page, u should have USER/ADMIN authority.
